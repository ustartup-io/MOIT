# План апгрейда сайта TERRARIUM-MSK

## 1. Аудит текущего сайта (terrariummsk.netlify.app)

1.  **Визуальный стиль и премиальность:** Текущий дизайн сайта функционален, но не передает ощущение эксклюзивности и премиальности, что критично для целевой аудитории. Цветовая палитра и типографика выглядят базовыми.
2.  **Недостаток высококачественного визуального контента:** Раздел "Наши работы" содержит лишь несколько примеров. Для демонстрации уникальности и качества продукта необходимо больше профессиональных фотографий или видеороликов.
3.  **Общие призывы к действию (CTA):** Кнопки "Рассчитать стоимость" и "Оставить заявку" стандартны и не отражают индивидуальный подход или ценность предложения.
4.  **Отсутствие четкого УТП на главной странице:** Хотя текст упоминает "эксклюзивные" террариумы, уникальное торговое предложение не выделено достаточно ярко и убедительно.
5.  **Базовое представление контактной информации:** Контакты в футере представлены просто. Более интегрированный и визуально привлекательный раздел контактов или форма могли бы улучшить конверсию.
6.  **Отсутствие отзывов и социального доказательства:** Для продукта с высоким средним чеком и индивидуальным подходом, отзывы клиентов или кейсы значительно повысили бы доверие и подтвердили премиальный статус.
7.  **Ограниченная демонстрация процесса создания:** Отсутствие информации о процессе работы над террариумом (от эскиза до установки) упускает возможность подчеркнуть ручную работу и индивидуальный подход.

## 4. План загрузки в GitHub (ustartup-io/MOIT)

Предполагается, что репозиторий `ustartup-io/MOIT` содержит структуру проекта Next.js/React.

1.  **Клонирование репозитория:**
    ```bash
    gh repo clone ustartup-io/MOIT moit_site
    cd moit_site
    ```
2.  **Создание новой ветки:**
    ```bash
    git checkout -b feature/terrarium-upgrade
    ```
3.  **Копирование файлов:**
    - Скопировать новые компоненты (`ContactForm.js`, `Testimonials.js`, `HowItWorks.js`, `FloatingMessengerButton.js`) в `moit_site/components/`.
    - Скопировать новую страницу `faq.js` в `moit_site/pages/`.
    - Скопировать новые изображения в `moit_site/public/images/`.
    - Обновить существующие файлы (`pages/index.js`, `components/Layout.js`, `pages/catalog.js`, `next.config.js`) в соответствующих директориях.
4.  **Проверка изменений:**
    - Убедиться, что все изменения внесены корректно и сайт локально собирается без ошибок.
5.  **Добавление, коммит и пуш изменений:**
    ```bash
    git add .
    git commit -m "feat: Implement terrarium website upgrade plan"
    git push origin feature/terrarium-upgrade
    ```
6.  **Создание Pull Request:**
    - Открыть Pull Request в репозитории `ustartup-io/MOIT` из ветки `feature/terrarium-upgrade` в `main` (или `dev`).

## Дополнительно

### 3 варианта заголовка + подзаголовка для главной (премиум-стиль, Москва)

1.  **Заголовок:** Террариумы, созданные для вашей уникальной экосистемы.
    **Подзаголовок:** Эксклюзивные проекты ручной работы в Москве, где каждая деталь имеет значение.
2.  **Заголовок:** Искусство живого пространства.
    **Подзаголовок:** Премиальные террариумы на заказ в Москве: воплощаем ваши мечты о совершенном биотопе.
3.  **Заголовок:** Ваш мир, созданный природой.
    **Подзаголовок:** Индивидуальные террариумы и вивариумы в Москве, гармонично интегрированные в ваш интерьер.

### 5 пунктов «Риск-контроль»

1.  **Несоответствие визуального контента:** Новые фотографии могут не соответствовать заявленному премиум-уровню, что снизит эффект от редизайна. **Решение:** Привлечение профессионального фотографа, строгий отбор изображений.
2.  **Проблемы с производительностью:** Добавление большого количества изображений и новых компонентов может замедлить загрузку сайта. **Решение:** Оптимизация изображений, ленивая загрузка (lazy loading), проверка Lighthouse.
3.  **Некорректная работа форм:** Форма заявки может не отправляться или данные могут теряться. **Решение:** Тщательное тестирование формы после внедрения, настройка уведомлений.
4.  **Нарушение мобильной адаптации:** Изменения в дизайне и верстке могут привести к некорректному отображению на некоторых мобильных устройствах. **Решение:** Кросс-браузерное и кросс-девайсное тестирование.
5.  **Недостаточное SEO:** Новые тексты и структура могут не принести ожидаемого SEO-эффекта без должного анализа ключевых слов и технической оптимизации. **Решение:** Использование инструментов для SEO-анализа, мониторинг позиций.

### Визуальные рекомендации

Для усиления премиального стиля рекомендуется использовать глубокие, насыщенные цвета (например, темно-зеленый, антрацит, золото) в сочетании с нейтральными оттенками. В типографике стоит отдать предпочтение элегантным шрифтам с засечками для заголовков и чистым, хорошо читаемым шрифтам без засечек для основного текста. Фотографии должны быть высокого разрешения, с акцентом на детали и естественное освещение, демонстрирующие красоту и уникальность каждого террариума. Минималистичный дизайн с большим количеством "воздуха" поможет создать ощущение роскоши и простора.
