# TERRARIUM-MSK Premium Website

Премиальный сайт студии эксклюзивных террариумов на заказ в Москве. Разработан для максимизации лидогенерации и демонстрации премиального позиционирования.

## 🎯 Реализованные шаги апгрейда

1. **Hero-секция** — высококачественное изображение террариума с асимметричным макетом
2. **Раздел услуг** — 6 ключевых преимуществ с иконками и описаниями
3. **Портфолио** — галерея реализованных проектов с названиями и размерами
4. **Процесс работы** — пошаговое описание 4 этапов создания террариума
5. **Отзывы клиентов** — социальное доказательство с рейтингами
6. **Форма заявки** — встроенная форма обратной связи с валидацией
7. **WhatsApp интеграция** — плавающая кнопка для быстрого контакта
8. **Премиальная типографика** — Playfair Display (заголовки) + Poppins (текст)
9. **Цветовая палитра** — глубокий лесной зелёный + золото + кремовый
10. **Мобильная адаптация** — полностью адаптивный дизайн для всех устройств

## 🎨 Дизайн-философия

**Organic Minimalism** — биофильный модернизм, сочетающий природные мотивы с чистым современным дизайном.

### Ключевые элементы
- **Цвета:** Глубокий зелёный (`oklch(0.35 0.1 145)`), кремовый, muted gold
- **Типографика:** Playfair Display (serif) для заголовков, Poppins (sans-serif) для текста
- **Макет:** Асимметричные секции, обильное белое пространство
- **Интерактивность:** Гладкие переходы (200-250ms), масштабирование кнопок на клик

## 🚀 Быстрый старт

### Установка зависимостей
```bash
pnpm install
```

### Локальная разработка
```bash
pnpm dev
```

Сайт будет доступен на `http://localhost:3000`

### Сборка для продакшена
```bash
pnpm build
```

## 🎯 Ключевые метрики

- **Hero-секция:** Привлекает внимание премиальным изображением и асимметричным макетом
- **CTA конверсия:** Две кнопки (основная + вторичная) в Hero и форма в контактной секции
- **Социальное доказательство:** 3 отзыва с рейтингами для повышения доверия
- **Мобильность:** 100% адаптивность для всех устройств (375px - 1920px)

## 🎨 Цветовая палитра

| Назначение | OKLCH | Описание |
|---|---|---|
| Primary | `oklch(0.35 0.1 145)` | Глубокий лесной зелёный |
| Accent | `oklch(0.65 0.12 60)` | Muted gold для акцентов |
| Background | `oklch(0.98 0.001 0)` | Кремовый белый |
| Foreground | `oklch(0.15 0.02 250)` | Глубокий уголь |

## 🔤 Типографика

- **Заголовки:** Playfair Display, 700 weight
  - H1: 3.5rem
  - H2: 2.5rem
  - H3: 1.875rem
- **Текст:** Poppins, 400-500 weight
  - Body: 1rem
  - Secondary: 0.875rem

## 📱 Адаптивность

- **Mobile:** 375px - 640px
- **Tablet:** 640px - 1024px
- **Desktop:** 1024px+

Все секции адаптируются для оптимального отображения на каждом размере экрана.

## 🔗 Интеграции

- **WhatsApp:** Плавающая кнопка с ссылкой на номер +7 (495) 123-45-67
- **Форма заявки:** Валидация на клиенте, уведомления через Sonner
- **Аналитика:** Встроена поддержка Umami Analytics

## 📝 Контент

Все текстовые материалы на русском языке, оптимизированы для московского рынка и премиального позиционирования.
